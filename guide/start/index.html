1. 安装¶

1.1 环境要求¶

  • Python 3.11 或更高版本
  • pip 包管理器

1.2 安装步骤¶

pip install my-project-doc

2. 基本使用¶

2.1 初始化项目¶

from my_project_doc import main

main()

2.2 配置文件¶

项目配置文件位于根目录的 pyproject.toml：

[project]
name = "my-project-doc"
version = "0.1.0"
description = "项目文档"

3. 常见问题¶

3.1 导入错误¶

如果遇到导入错误，请检查： - Python 版本是否符合要求 - 包是否正确安装 - 虚拟环境是否激活

3.2 运行错误¶

常见运行错误及解决方案： 1. 配置文件不存在 2. 权限问题 3. 依赖缺失
