项目简介¶

my-project-doc 是一个用于测试和学习 MkDocs 文档生成工具的示例项目。

技术栈¶

  • Python 3.11
  • MkDocs
  • Material for MkDocs 主题
  • Markdown

联系方式¶

如有问题或建议，请通过以下方式联系：

  • GitHub Issues
  • Email: your.email@example.com

贡献指南¶

欢迎提交 Pull Request 或提出 Issue。在贡献代码前，请确保：

  1. 代码符合项目规范
  2. 添加必要的测试
  3. 更新相关文档
